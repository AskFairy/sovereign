明白，你要的是“代码层面我到底怎么做的”。核心不是调接口，而是浏览器自动化读取 DOM：

```text
打开题库页
-> 找到题库链接
-> 进入题库详情页
-> 找到第一道题链接
-> 进入题目页
-> 读取标题/难度/标签/正文
-> 点击 tab
-> 读取 tab 内容
-> 生成 Markdown
```

对应成代码大概是这样：

```js
// 1. 打开题库列表页
await page.goto("https://www.mianshiya.com/banks");
await page.waitForLoadState("domcontentloaded");

// 2. 找第一个题库链接
const bankUrl = await page.evaluate(() => {
  const a = document.querySelector('a[href^="/bank/"]');
  return new URL(a.getAttribute("href"), location.origin).href;
});

// 3. 进入题库页
await page.goto(bankUrl);
await page.waitForLoadState("domcontentloaded");

// 4. 找第一道题链接
const questionUrl = await page.evaluate(() => {
  const a = document.querySelector('a[href*="/question/"]');
  return new URL(a.getAttribute("href"), location.origin).href;
});

// 5. 进入题目详情页
await page.goto(questionUrl);
await page.waitForLoadState("domcontentloaded");
```

然后读取题目的基础信息：

```js
const meta = await page.evaluate(() => {
  const clean = (s) => (s || "").replace(/\s+/g, " ").trim();

  const title = clean(document.querySelector("h1")?.textContent);

  const tags = [...document.querySelectorAll('a[href^="/tag/"]')]
    .map((a) => clean(a.textContent));

  const text = document.body.innerText;

  const difficulty =
    text.includes("中等") ? "中等" :
    text.includes("简单") ? "简单" :
    text.includes("困难") ? "困难" :
    "";

  return {
    title,
    tags,
    difficulty,
    url: location.href,
  };
});
```

读取「推荐答案」区域，我是从当前选中的 tab panel / 主内容区域取可见文本，然后再整理成摘要：

读取图片链接：

```js
const imageUrls = await page.evaluate(() => {
  const main = document.querySelector("main main") || document.querySelector("main");

  return [...main.querySelectorAll("img")]
    .map((img) => ({
      alt: img.getAttribute("alt") || "",
      src: img.currentSrc || img.src || "",
    }))
    .filter((img) =>
      img.src &&
      !/avatar|logo|star|eye|tag|share|ellipsis|level/i.test(
        img.alt + " " + img.src
      )
    );
});
```
